## 0️⃣ 메인 페이지

### 메인

- 맛집 키워드 검색 칸
- 검색 버튼
- 키워드 검색 하면 Naver API로 상세 정보 받아와서 출력
- 맛집 정보 출력하고 위시리스트 추가 버튼 표시
- 위시리스트 추가 시 위시리스트에 표시되게 함
- 위시리스트는 다른 키워드 검색 시에도 계속 표시
- 위시리스트에 방문 추가, 위시리스트 삭제 버튼 활성화
- 방문 추가 버튼 클릭시 방문 횟수 올라감
